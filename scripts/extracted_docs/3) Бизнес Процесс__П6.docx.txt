ХАРИДЛАР.УЗ БИЗНЕС АРХИТЕКТУРАСИ
6-БИЗНЕС ЖАРАЁНИ
МОЛИЯВИЙ ҲИСОБ-КИТОБЛАР, ҲИСОБ-ФАКТУРА ВА ТЎЛОВЛАРНИ БОШҚАРИШ
Ҳужжат тури: Бизнес жараёни (Business Process)
Жараён рақами: BP-06
Версия: 2.0 (Таҳрирланган)
1. Жараённинг мақсади
BP-05 жараёнида мижозга муваффақиятли етказиб берилган товар ёки хизмат асосида молиявий ҳужжатларни расмийлаштириш, ҳисоб-фактураларни шакллантириш, мижоз томонидан қабул қилинишини таъминлаш, шартнома шартларига мувофиқ тўловларни ўз вақтида ундириш ҳамда мижоздан келиб тушган маблағлар ҳисобидан таъминловчилар билан ҳисоб-китобларни белгиланган тартибда амалга ошириш.
Ушбу жараённинг асосий вазифаси — бажарилган хизмат ва етказиб берилган маҳсулотни пул оқимига (Cash Flow) айлантириш, шу билан бирга таъминловчилар олдидаги молиявий мажбуриятларни ўз вақтида бажариш.
2. Жараённинг бизнесдаги ўрни
Бу жараён компаниянинг даромадни реализация қилиш ва молиявий мажбуриятларни бошқариш босқичидир.
Энг яхши сотув, энг яхши тендер ва энг яхши логистика ҳам, агар:
мижоздан пул ўз вақтида келмаса;
таъминловчилар билан ҳисоб-китоб кечикса;
компаниянинг пул оқими (Cash Flow) ва бозордаги ишончлилигига салбий таъсир кўрсатади.
Шу сабабли ушбу жараён икки асосий мақсадга хизмат қилади:
Мижоздан маблағни ўз вақтида ундириш.
Мижоздан маблағ келиб тушиши билан таъминловчига тўловни белгиланган тартибда амалга ошириш.
Бу орқали компаниянинг ҳам мижозлар, ҳам таъминловчилар олдидаги ишончлилиги таъминланади.
3. Жараённинг асосий натижаси (Output)
Жараён якунида:
✓ Ҳисоб-фактура расмийлаштирилган.
✓ Ҳисоб-фактура мижоз томонидан қабул қилинган.
✓ Мижоздан тўлов қабул қилинган.
✓ Мижоз тўлови битим билан боғланган.
✓ Таъминловчига шартнома шартларига мувофиқ тўлов амалга оширилган.
✓ Таъминловчи билан ҳисоб-китоб ёпилган.
✓ Компаниянинг даромади ва маржаси ҳисобланган.
✓ Битим молиявий жиҳатдан тўлиқ якунланган.

маблағ тушган сана қайд этилади.
9-босқич. Таъминловчи билан ҳисоб-китоб қилиш
Мижоздан маблағ келиб тушганлиги тасдиқлангач:
таъминловчи олдидаги қарздорлик текширилади;
шартнома ва келишув шартлари асосида тўлов суммаси тасдиқланади;
тўлов топшириқномаси тайёрланади;
банк орқали таъминловчига тўлов амалга оширилади;
тўловнинг бажарилганлиги тасдиқланади;
таъминловчи билан ҳисоб-китоб ёпилади.
Асосий қоида:
Агар таъминловчи билан бошқача келишув мавжуд бўлмаса, мижоздан маблағ келиб тушган куннинг ўзида ёки кейинги банк иш кунида таъминловчига тўлов амалга оширилиши шарт.
Бу компаниянинг ҳамкорлар билан узоқ муддатли ишончли муносабатларини сақлашнинг асосий тамойилларидан бири ҳисобланади.
10-босқич. Битимни молиявий ёпиш
Қайд қилинади:

маржа ҳисобланди;
дебитор ва кредитор қарздорлик ёпилди;
битим молиявий жиҳатдан тўлиқ якунланди.
Бу билан BP-06 жараёни якунланади.
8. Қарор қабул қилиш нуқталари (Decision Points)
DP-01
Ҳужжатлар тўлиқми?
Ҳа → Ҳисоб-фактура тайёрланади.
Йўқ → Камчиликлар бартараф қилинади.
DP-02
Мижоз ҳисоб-фактурани қабул қилдими?
Ҳа → Тўлов мониторинги.
Йўқ → Ҳужжатлар тузатилади.
DP-03
Тўлов келиб тушдими?
Ҳа → Таъминловчи билан ҳисоб-китоб жараёни бошланади.
Йўқ → Дебитор қарздорлик жараёни давом этади.
DP-04
Таъминловчига тўлов амалга оширилдими?
Ҳа → Битим молиявий жиҳатдан ёпилади.
Йўқ → Сабаби аниқланади ва зудлик билан бартараф қилинади.

16. NEXT ONE эксперт хулосаси
Бу ўзгартишдан кейин BP-06 жараёни анча тўлиқ ва реал бизнес моделига мос бўлди.
Мен яна бир муҳим таклифни ҳам киритишни тавсия қиламан.
Ҳозирги ҳужжатда молиявий жараён мижоз → компания → таъминловчи кетма-кетлигида қурилган. Бу тўғри. Лекин Харидлар.уз бизнес моделининг ўзига хос жиҳати шундаки, компания айланма маблағни самарали бошқариш орқали юқори тезликда кўплаб битимларни амалга оширади.
Шу сабабли BP-06 ичида "Тўловлар устуворлиги сиёсати" (Payment Priority Policy) алоҳида ички регламент сифатида ишлаб чиқилиши мақсадга мувофиқ.
Унда қуйидаги устуворликлар белгиланади:
Қонунчиликда белгиланган мажбурий тўловлар.
Мижоздан маблағи келиб тушган битимлар бўйича таъминловчиларга тўловлар.
Стратегик таъминловчилар билан ҳисоб-китоблар.
Операцион харажатлар.
Бошқа молиявий мажбуриятлар.
Бу тартиб компаниянинг таъминловчилар билан ишончли муносабатларини сақлайди, таъминот занжирини узлуксиз қилади ва келгуси битимларда ҳамкорларнинг ишончини мустаҳкамлайди. Ушбу қўшимча BP-06 ни нафақат бухгалтерия жараёни, балки пул оқими ва ҳамкорлик ишончини бошқариш жараёни даражасига олиб чиқади.
