ЛИД ГЕНЕРАЦИЯСИ ВА БУЮРТМАНИ ШАКЛЛАНТИРИШ (Буюртма олишни бошлаш)
Ҳужжат тури: Бизнес жараёни (Business Process)
Жараён рақами: BP-01
Версия: 1.0 (Дастлабки таҳрир)
1. Жараённинг мақсади
Харидлар.уз компанияси учун янги мижозларни жалб қилиш, улар билан ишончли алоқа ўрнатиш, эҳтиёжларини аниқлаш, буюртмаларни тўлиқ шакллантириш ва кейинги жараёнларга тайёр ҳолда узатиш.
Ушбу жараённинг асосий вазифаси — тасодифий мурожаатни аниқ техник талабларга эга бўлган расмий буюртмага айлантиришдир.
2. Жараённинг бизнесдаги ўрни
Бу жараён компаниянинг барча савдо занжирининг бошланғич нуқтаси ҳисобланади.
Ушбу босқичдаги ҳар қандай хатолик кейинги барча жараёнларга таъсир қилади:
нотўғри тижорий таклиф тайёрланиши;
нотўғри нарх шаклланиши;
тендерда мағлубият;
логистикада муаммолар;
тўлов кечикиши;
мижоз норозилиги.
Шу сабабли ушбу жараён компаниянинг энг муҳим жараёнларидан бири ҳисобланади.
3. Жараённинг асосий натижаси (Output)
Жараён якунида қуйидаги натижа олиниши керак:
✓ Мижоз билан алоқа ўрнатилган.
✓ Мижоз эҳтиёжи тўлиқ аниқланган.
✓ Техник талаблар расмийлаштирилган.
✓ Савдо имконияти баҳоланган.
✓ Буюртма тасдиқланган.
✓ Буюртма Нархлаш бўлимига тижорий таклиф тайёрлаш учун юборилган.

Ҳа →

Йўқ →

DP-03

Ҳа →

Йўқ →
Мижозга рад жавоби ёки муқобил ечим таклиф қилинади.

16. NEXT ONE эксперт хулосаси
Ҳозирги ҳолатда Харидлар.уз компаниясида сотув жараёни мавжуд, аммо у асосан ходимлар тажрибасига таянган ҳолда ишлайди. Бу эса компания ўсиши билан маълумот йўқолиши, турлича ишлаш услублари ва назоратнинг мураккаблашишига олиб келиши мумкин.
Ушбу ҳужжатнинг жорий қилиниши натижасида:
ҳар бир буюртма ягона стандарт асосида қабул қилинади;
мижоз ҳақидаги маълумотлар тўлиқ сақланади;
кейинги жараёнларга сифатли маълумот узатилади;
KPI асосида бошқариш имконияти пайдо бўлади;
CRM ва ERP тизимларини жорий этиш учун мустаҳкам асос яратилади.
